### 1. **Ask Questions & Share Ideas**
- Use [GitHub Discussions](https://github.com/mrlynn/create-mongonext-app/discussions) for questions, ideas, and community support.

### 2. **Report Bugs**
- Search [existing issues](https://github.com/mrlynn/create-mongonext-app/issues) before opening a new one.
- Include clear steps to reproduce, error messages, and your environment (OS, Node.js version, etc).

### 3. **Suggest Features**
- Open a new issue with the `feature request` label.
- Describe your use case and why it would benefit the project.

### 4. **Submit Pull Requests**
- Fork the repo and create a new branch for your change.
- Write clear, concise commit messages.
- Add tests for new features or bug fixes if possible.
- Run `npm install` and `npm run lint` to ensure code quality.
- Open a pull request and describe your changes.
